# Partners

ts-analytics integrates with leading tools and platforms.

## Technology Partners

### AWS

ts-analytics is built on AWS services:
- DynamoDB for storage
- Lambda for serverless compute
- CloudFront for CDN

### Cloudflare

Deploy analytics at the edge with Cloudflare Workers.

### Stacks

First-class integration with the Stacks framework.
